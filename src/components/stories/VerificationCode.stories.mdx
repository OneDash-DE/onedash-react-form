## Loading

If you provide a loading spinner and the input state is set to disabled, the spinner will be shown.

```jsx
const Spinner = (
	<div class="example-spinner">
		<div class="bounce1"></div>
		<div class="bounce2"></div>
		<div class="bounce3"></div>
	</div>
);
// Use this Spinner now as loading Spinner
<VerificationCode name="verification-code" codeLength={4} loadingSpinner={Spinner} disabled />;
```
