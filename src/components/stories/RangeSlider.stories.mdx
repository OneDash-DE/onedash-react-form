# Input

The range slider allowes to select either one value or a range of a number range

## Default

```jsx
<RangeSlider name="default-range-slider" minValue={0} maxValue={10} defaultValue={5} />
```

## Format Label

```jsx
<RangeSlider
	name="default-range-slider"
	formatLabel={(val) => ["A", "B", "C", "D", "E", "F"][val]}
	minValue={1}
	maxValue={5}
	defaultValue={{ min: 2, max: 3 }}
/>
```
